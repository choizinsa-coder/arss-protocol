# AIBA 프로젝트 공식 용어집 (Glossary)

제정 세션: S436 | 승인: EAG-S436-DOCFIX-GLOSSARY-001
제정 사유: S436에서 "사업 착수"를 서로 다른 의미로 사용하여
           헌법 원칙 위반 여부를 놓고 반나절의 혼선이 발생했다.

원칙: 모든 문서는 본 용어집의 정의를 기준으로 해석한다.
      용어집과 다른 의미로 쓰려면 그 문서에 명시적으로 다르게 정의해야 한다.

---

## 사업 4단계 (S436 확정)

| 단계 | 정의 | 게이트 |
|------|------|--------|
| 사업 정의 | 무엇을 누구에게 팔 것인지 결정 | 없음 (준비) |
| 사업 준비 | 기능 설계, 고객 탐색, 시범 대상 확보 | 없음 (준비) |
| 시범 운영(Pilot) | 제한된 실제 환경에서 검증 | 비오님 승인 필요 |
| 사업 착수 | 실제 고객에게 서비스를 제공하고 운영 시작 | 안정 1단계 완성 + 비오님 승인 |

주의: S416 로드맵의 "비즈니스 착수 금지"는 위 표의 **사업 착수**를 가리킨다.
      사업 정의·사업 준비는 금지 대상이 아니다.

## 판독 상태 용어 (S436 확정)

| 용어 | 의미 |
|------|------|
| Observed (관찰) | 문서 일부만 확인한 잠정 상태. 확정 표현 사용 금지 |
| Confirmed (판정) | 전문 대조 완료. 확정 표현 사용 가능 |

## 확정 표현 (Confirmed 단계에서만 사용)

충돌 / 위반 / 무효 / 철회 / 완료 / 정상
