C1. 계약서 주요 조항 검토 지원
 산출물: 업로드된 계약서(PDF/Word)에서 책임 조항·배상 한도·해지 조건·기밀 유지 등 핵심 조항을 추출하고 위험 수준을 표시한 검토 리포트
 사용자: 변호사·법무팀 담당자 (실제 계약 검토 업무 수행자)
 최소 구현: 파일 업로드 → 조항 추출 및 위험 태깅 → 변호사가 검토·수정·서명으로 승인하는 단일 워크플로우

C2. 법령·판례 리서치 보조
 산출물: 자연어 질문(예: "하자담보책임 기간이 어떻게 되나요")에 대응하는 관련 법령 조문·판례 목록과 3문장 요약
 사용자: 변호사·사무장 (법적 근거 확인이 필요한 모든 업무 담당자)
 최소 구현: 질문 입력 → 내부 법령 DB(또는 공개 API) 검색 → 결과 요약 + 인용 출처 표시 → 변호사 확인 후 인용 확정

C3. 소송·계약 기한 자동 관리
 산출물: 사건별(소송·계약·등기) 법정 기한과 내부 마감일을 자동 산출한 타임라인 대시보드 + 기한 임박 알림
 사용자: 변호사·법률 비서 (기한 도과 시 손해 배상 책임을 지는 실무자)
 최소 구현: 사건 기본 정보 입력 → 법령 기준 기한 자동 계산 → 달력 표시 + N일 전 알림 → 변호사 확인으로 기한 확정
